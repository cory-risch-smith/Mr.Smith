Kubernetes

Dev-Ops | November 4, 2020

...continued

One of the things I started to look a little bit more into is a technology that was developed by Google and released in 2015 called Kubernetes. Kubernetes is an open-source container orchestration system that can be used for a variety of different things, including app deployment, scaling, and management. What makes this system so powerful is that it can essentially act as a virtual machine, a virtual computer that works alongside your own. The reason this is so interesting is the way that Kubernetes works. Kubernetes are composed of what are called pods, which are little containers where you can keep files, programs, and allow them to run solely from there. Think of it as a storage unit with your program in it. The great thing about using these containers is that you can link your local server to the same server the pod will be operating on, freeing up your computer the space and not having it bogged down. The cool part is, you can literally run anything from these containers; games, programs, and they are pretty secure too. They are a new and trending technology in the cloud architecture space.

- Cory Risch-Smith
